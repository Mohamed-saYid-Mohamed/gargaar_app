# Gargaar App

Gargaar App is a Flutter-based cross-platform application for reporting incidents, managing records, handling notifications, and connecting users with Firebase services.

## Features

- Firebase authentication
- Incident reporting and tracking
- Chat support
- Notifications
- Profile management
- Records and history screens
- Supports Android, iOS, Web, Windows, macOS, and Linux

## Tech Stack

- Flutter
- Dart
- Firebase Auth
- Cloud Firestore
- Firebase Storage
- Firebase Messaging
- FlutterFire CLI

## Project Setup

### Requirements

- Flutter SDK
- Dart SDK
- Firebase project
- Android Studio, Xcode, or VS Code

### Install dependencies

```bash
flutter pub get
flutterfire configure
Running the app
Android / iOS / Desktop
flutter run
